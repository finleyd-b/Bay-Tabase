# Bay-Tabase: A Shiny Database Explorer

## Overview
Bay-Tabase is an interactive Shiny app that lets users explore, filter, and visualize marine occurrence and environmental data in the San Francisco Bay through an intuitive interface. The app demonstrates skills in data management, statistical exploration, and interactive visualization within a reproducible workflow, and it's designed to be accessible to users both inside and outside the scientific or statistical fields.

## Motivation
Marine datasets can be large and complex. Bay-Tabase makes these data more accessible by providing dynamic filters, maps, and summary visualizations that allow users to explore patterns in species distributions and environmental variables.
